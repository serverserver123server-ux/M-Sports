# 🏎️ BMW Carbon & Crimson Edition (M-Sport)

An aggressive, ultra-sleek dark theme for Discord inspired by the iconic **BMW M-Performance** division. Built on a heavily optimized, completely standalone engine block to guarantee maximum speed, fluidity, and custom aesthetics.

---

## 🛠️ Performance Features & Modifications

This build drops unnecessary background weight and tunes the UI from the ground up:

* **Flawless Fullscreen Scaling:** Hard-coded `cover` attributes prevent background stretching or flattening across various window aspect ratios.
* **Custom Status Array:**
    * 🔴 **Online:** Crimson Red
    * 🔵 **Idle:** BMW M Light Blue
    * 🟢 **Do Not Disturb:** Clean Track Green
    * 🔮 **Offline/Invisible:** Rich Midnight Blue (`#290066`)
* **De-Bloated Interface:** Stripped out Discord retail bloat including Nitro storefront ads, Shop tabs, Gift buttons, and Discover grids.
* **High-End Glassmorphism:** Mini-profile popouts feature transparent acrylic cards with a `12px` heavy backdrop blur filter.
* **Dynamic Animations:** Interactive message slide-ins, glow rings for active voice channel speakers, and responsive bouncy avatars on hover.
* **Custom Workspace Branding:** Obliterated original credit lines to run an official `M-Sport` tracking system directly in the window panel.

---

To apply this setup to your client, copy the configuration block below and paste it directly into your local **Vencord** or **BetterDiscord** custom themes tab.

 * @source [https://github.com/serverserver123server-ux/M-Sports](https://github.com/serverserver123server-ux/M-Sports)
